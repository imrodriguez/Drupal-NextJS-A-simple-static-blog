# A enviroment of JAM Stack with Drupal and ReactJS on Docker

Technologies
========
  - Drupal
  - Apache
  - PHP
  - ReactJS
  - Docker

Configuration
========
  - Copy .env.example to .env and modify it if needed. 

Install
========
  - Execute install.sh

Start
========  
  - Build containers with docker-compose build
  - Start containers with docker-compose up
